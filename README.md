# 📺 YouTube Video List

Este é um aplicativo React criado com **Vite** que exibe uma lista de vídeos incorporados do YouTube. É uma maneira simples e rápida de reunir vídeos favoritos em um só lugar.

## ✨ Funcionalidades

- Exibição de vídeos do YouTube com `<iframe>`
- Suporte a links `embed`, `shorts` e `youtu.be`
- Estilização moderna e responsiva
- Publicação fácil no GitHub Pages

## 🚀 Deploy

Acesse a versão online aqui (exemplo):
[https://seu-usuario.github.io/youtube-video-list](https://seu-usuario.github.io/youtube-video-list)

## 🔧 Tecnologias

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [GitHub Pages](https://pages.github.com/)

## 📦 Como rodar localmente

1. Clone o repositório:

   ```bash
   git clone https://github.com/seu-usuario/youtube-video-list.git
   cd youtube-video-list
   ```

2. Instale as dependências:

   ```bash
   npm install
   ```

3. Rode o projeto localmente:

   ```bash
   npm run dev
   ```

4. Acesse no navegador:
   ```
   http://localhost:5173
   ```

## 📤 Deploy no GitHub Pages

1. Altere o arquivo `vite.config.js` com o nome do repositório:

   ```js
   base: '/youtube-video-list/'
   ```

2. Faça build e deploy:

   ```bash
   npm run build
   npm run deploy
   ```

> Requer `vite-plugin-gh-pages` e um arquivo `deploy.js`.
